# Security Policy

## Supported Versions

We provide security updates for the current major version of Synaptiq. Please ensure you are running the latest stable release.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| 0.x.x   | :x:                |

## Reporting a Vulnerability

We take security issues very seriously. If you discover a security vulnerability within Synaptiq, please **do not open a public issue**. 

Instead, send an email to **privacy@spectrayan.com** and **legal@spectrayan.com** (if applicable) or reach out to our support team at **support@spectrayan.com**.

Please include:
* The type of vulnerability (e.g., XSS, SQLi, IDOR)
* Steps to reproduce the issue
* Any potential impact you have identified

We will aim to acknowledge your report within 48 hours and will provide a status update as we investigate and develop a fix.

## Security Disclosures

Once a vulnerability has been confirmed and a fix is ready, we will publish a security advisory and notify the community via our standard release channels. We kindly ask that you keep the details confidential until the advisory is published to protect our users.
