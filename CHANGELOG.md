# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Multi-step conversational AI workflows powered by Spring AI.
- Migration to Spring Boot 4 for backend microservices.
- Contributor Covenant Code of Conduct and Security Policy.

### Changed
- Open-sourced the repository under the Apache 2.0 License.
- Deprecated Python FastAPI backend in favor of Java Spring Boot backend.

### Removed
- Python FastAPI dependencies and backend code.

## [1.0.0] - 2026-04-20

### Added
- Initial stable release of the Synaptiq AI Orchestration Platform.
- AI-generated dynamic components and Component DSL schema.
- Vector search with MongoDB Atlas.
- Angular 21 frontend shell.
